This is a repository to record my HW &amp; Project for Computational Physics A

## HW1
用 Schrage 方法编写随机数子程序，用指定间隔（非连续 $l >1$）两个随机数作为点的坐标值绘出若干点的平面分布图。再用 $<x^k>$ 测试均匀性（取不同量级的 $N$ 值，讨论偏差与 $N$ 的关系）、$C(l)$测试其 2 维独立性（总点数 $N > 10^7$）。

## HW2
